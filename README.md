# HarmonyOSQ&A

华为开发者网站智能客服 CLI 工具 - 通过调用华为开发者官网 AI 客服 API，实现交互式流式问答。

## 功能特性

- 交互式问答，支持连续对话
- SSE 流式响应 + 打字机效果，实时逐字输出
- HTTP/2 协议连接，确保流式数据不被 CDN 缓冲
- 自动提取参考来源链接
- 支持命令行单次提问和交互模式

## 安装依赖

```bash
pip install httpx
```

## 使用方法

### 命令行单次提问

```bash
python qa.py float.json里可以写24vp吗
```

### 交互模式

```bash
python qa.py
```

进入交互模式后，输入问题即可对话，输入 `q` / `quit` / `exit` 退出。

## 技术说明

脚本使用 `httpx` 库的 HTTP/2 模式连接华为开发者网站的智能客服 API。这是因为该网站的 CDN/反向代理对 HTTP/1.1 的 SSE 流会做整段缓冲，导致数据最后才一次性返回；而 HTTP/2 下 SSE 流直接透传，数据到达即可处理。

输出的打字机效果速度可通过代码中 `TYPEWRITER_DELAY` 参数调整（默认 0.02 秒/字符）。
